# MarioScene

### KeyBinding

| Key          	| Action      	                    |
|--------------	|---------------------------------  |
| W            	| Forward     	                    |
| S            	| Backward    	                    |
| A            	| Left        	                    |
| D            	| Right       	                    |
| R            	| Up          	                    |
| F            	| Down        	                    |
| Escape       	| Close window                      |
| Enter        	| Change post processing effect     |
| Space       	| Draw with or without empty faces  |
| Mouse scroll 	| Zoom in/out 	                    |
